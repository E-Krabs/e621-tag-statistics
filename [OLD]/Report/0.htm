e621 Data

About

Scripts to analyze the furry fandom through the content it produces and consumes. Every image on e621 must be tagged with info describing what's in it (characters, artist, acts, etc.). Using this information provided via the e621 API, we can plot the popularity of something in furry media.

Fetching Data

This project fetches data from e621 provided via the e621 API. fetchall.py collects as much data as it can (~2.5mil posts). This opperation can take a few hours, because API requests are limited to 2 per second. But we make requests only once per second to avoid a 503 (too many requests). Every request returns 320 posts. The script writes the returned results to a JSON file (~4.5GB) for later analysis.

How It Works

Everything is written in Python. We fetch our information using the requests library and save it as JSON. We can then use pandas to create a dataframe from the JSON objects. We then plot the dataframe with matplotlib.
